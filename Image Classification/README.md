# Image Classification with CNN

This project demonstrates a simple Convolutional Neural Network (CNN) for image classification using Python and TensorFlow.

## Project Overview

The goal of this project is to build and train a CNN model for classifying images.  
This notebook demonstrates the full workflow:

- Data loading
- Image preprocessing
- CNN architecture
- Model training
- Evaluation

## Technologies

- NumPy
- Matplotlib
- Jupyter Notebook
